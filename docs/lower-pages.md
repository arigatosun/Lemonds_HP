この資料は、トップページ以外のページについて、制作対象となるページ構成と表示内容を整理したものです。
PDF上で本文が見えている箇所は具体文言まで記載し、仮文・空欄・未確定箇所はそのまま明示しています。

## 前提

本構成で必要な下層ページは以下です。

| ページ | 想定パス | 役割 |
|---|---|---|
| 事業内容 | `/services` | 8事業の詳細を見せるページ |
| 制作実績一覧 | `/works` | 実績を一覧で見せるページ |
| 制作実績詳細 | `/works/{slug}` | 個別実績の詳細を見せるページ |
| ニュース一覧 | `/news` | お知らせを一覧で見せるページ |
| ニュース詳細 | `/news/{slug}` | 個別のお知らせ本文を見せるページ |
| 会社概要 | `/company` | 会社紹介、会社概要、理念、所在地を見せるページ |
| お問い合わせ | `/contact` | 問い合わせ・見積もり依頼フォーム |
| 送信完了ページ | `/contact/thanks` | フォーム送信後の完了ページ |
| ポリシーページ | `/policy` または `/privacy` | 企業ポリシー、個人情報保護方針、反社方針を掲載 |

本構成では独立ページとして不要、または任意扱いとするページは以下です。

| ページ | 扱い |
|---|---|
| 弊社の強み / 選ばれる理由 | 独立ページは不要。強み要素を使う場合はTOPや事業内容内に要約配置する |
| 対談・インタビュー | 初期実装に含めるか要確認。必要な場合は制作実績配下または投稿タイプで扱う |
